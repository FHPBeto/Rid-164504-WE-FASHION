   WE FASHION - Escola de Moda 
 Status do Projeto: Finalizado para Apresentação | RID: 164504

   Sobre o Projeto
Este é um sistema Full Stack desenvolvido para a escola WE FASHION. O objetivo é capturar o interesse de novos alunos e oferecer um painel administrativo exclusivo para a gestão da escola.

   Tecnologias e Arquitetura
O projeto foi estruturado com separação de responsabilidades (Client/Server):

- Front-end (we-fashion): Desenvolvido em React + Vite e TypeScript. Possui design moderno com efeitos de transparência (Glassmorphism).
- Back-end (servidor-we-fashion): Servidor em Node.js e Express que gerencia o salvamento de dados em arquivos locais.
- Segurança: Acesso ao painel de administração protegido por senha.

   Como executar
Para rodar o projeto localmente:
1. Navegue até `we-fashion` e use `npm run dev`.
2. Navegue até `servidor-we-fashion` e use `node index.js`.
